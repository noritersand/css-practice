display: flex

개요

display: flex는 CSS의 Flexible Box Layout(줄여서 Flexbox) 모듈을 활성화하는 프로퍼티로, CSS 박스 모델의 레이아웃을 1차원으로 설정할 때 사용한다.

기존에 사용하던 float나 inline-box 같은 오래된 프로퍼티 비해 배치와 정렬이 매우 쉽다.

  • flex container: Flexbox Layout의 시작점이자 flex item들을 감싸는 부모 엘리먼트
  • flex item: flex container 내부에 배치되는 구성 요소이자 자식 엘리먼트
  • main axis: flex item들이 정렬되는 기본 진행 방향의 축
  • cross axis: main axis와 수직을 이루는 교차 방향의 축
