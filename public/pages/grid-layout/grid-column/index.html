grid-column

개요

grid-column은 그리드 컨테이너 내의 개별 아이템이 세로 라인(grid line)을 기준으로 어디서부터 어디까지의 column track 영역을 차지할지 지정한다.

grid-column-start와 grid-column-end를 한 번에 설정하는 단축 프로퍼티다.

그리드 아이템에 직접 지정한다. 기본값은 auto

  • grid-column-start: 아이템이 시작할 라인의 번호나 이름을 지정한다.

  • grid-column-end: 아이템이 끝날 라인의 번호나 이름을 지정한다.
