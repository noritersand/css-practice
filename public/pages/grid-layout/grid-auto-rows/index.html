grid-auto-rows

개요

grid-auto-rows는 그리드 컨테이너에서 명시적으로 행 트랙 크기를 정의하지 않았을 때 동적으로 생성되는 암시적 행의 크기를 설정하는 프로퍼티다.

grid-template-rows로 지정된 명시적 범위를 벗어나 그리드 아이템이 추가되거나, grid-row 등을 통해 명시적 영역 밖에 아이템을 배치할 때 자동으로 만들어지는 행에 적용된다.

이 프로퍼티는 그리드 컨테이너에 지정하며, 기본값은 auto다.

특이사항

grid-template-rows와의 차이

grid-template-rows는 몇 번째 행이 몇 px인지 미리 정의하는 명시적(Explicit) 설정이고, grid-auto-rows는 정의된 범위를 넘어선 아이템들이 유입될 때 적용되는 암시적(Implicit) 설정이다.

grid-auto-flow와의 관계

기본적으로 grid-auto-flow는 row로 설정되어 있어 아이템이 넘치면 아래쪽으로 새로운 행이 생성되므로 grid-auto-rows가 적용된다. 만약 grid-auto-flow: column으로 변경하여 열 방향으로 아이템이 늘어나게 설정한 경우에는 grid-auto-rows가 아닌 grid-auto-columns 프로퍼티가 작동한다.

음수 인덱스 배치 시 주의점

grid-row: -1 등의 음수 인덱스로 아이템을 배치하거나 명시적 트랙 상단(1번 라인 이전)으로 아이템을 밀어 넣을 때 생성되는 암시적 행에도 grid-auto-rows의 크기 규격이 동일하게 적용된다.

비율 단위(fr)는 컨테이너의 명시적인 높이(height)가 존재할 때 해당 잔여 공간을 지정된 비율로 분배한다.
