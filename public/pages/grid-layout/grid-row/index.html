grid-row

개요

grid-row은 Grid 컨테이너 내의 개별 아이템이 가로 라인(Grid Line)을 기준으로 어디서부터 어디까지의 Row Track 영역을 차지할지 지정한다.

grid-row-start와 grid-row-end를 한 번에 설정하는 단축 프로퍼티다.

Grid 아이템에 직접 지정한다. 기본값은 auto

  • grid-row-start: 아이템이 시작할 라인의 번호나 이름을 지정한다.

  • grid-row-end: 아이템이 끝날 라인의 번호나 이름을 지정한다.
