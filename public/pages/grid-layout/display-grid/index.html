display: grid

개요

display: grid는 CSS Grid Layout(줄여서 Grid) 모듈을 활성화하는 프로퍼티로, Grid Layout은 엘리먼트들을 2차원 그리드로 배치하는 데 사용한다.

Flexbox보다 제공되는 기능이 더 많고 강력하다.

  • Grid Item: Grid Container의 구성 요소
  • Grid Row Track: 행을 구성하는 가로 한 줄 영역 전체
  • Grid Column Track: 열을 구성하는 세로 한 줄 영역 전체
  • Grid Line: 행과 열을 나누는 경계선(트랙의 테두리).
  • Grid Cell: Grid Line으로 둘러싸인 가장 작은 단위 공간(칸)
  • Grid Area: 여러 Grid Cell로 구성된 영역. 반드시 직사각형 형태여야 한다. 가령 'L'자나 'T'자 형태의 Grid Area는 존재할 수 없다
