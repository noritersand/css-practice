grid-area

개요

grid-area는 그리드 컨테이너 내의 개별 아이템이 차지할 영역을 지정하는 단축 프로퍼티다.

grid-row-start, grid-column-start, grid-row-end, grid-column-end를 한 번에 설정하며, 각 프로퍼티는 슬래시로 구분한다.

그리드 아이템에 직접 지정한다. 기본값은 auto

Syntax

Values

어쩌구

auto

일부만 auto로 지정하거나 모든 프로퍼티를 auto로 지정할 수 있다.

프로퍼티를 생략하면 auto로 설정되기 때문에 아래 넷은 동일하게 작동한다:

grid-row-start, grid-column-start를 auto로 지정하면, 그리드 배치 알고리즘에 의해 알아서 빈 공간을 찾는다

grid-row-end, grid-column-end를 auto로 지정하면 딱 1개의 트랙만 차지한다. (span 1과 동일)
