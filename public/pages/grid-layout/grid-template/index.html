grid-template

개요

grid-template은 grid-template-rows, grid-template-columns, grid-template-areas를 한 번에 정의하는 단축 프로퍼티다.

Grid 컨테이너에 직접 지정하며, 기본값은 none이다.

프로퍼티별 특성은 각 페이지를 확인할 것.
