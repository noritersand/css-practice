grid-template-areas

개요

grid-template-areas는 Grid Area에 이름을 부여하고, 2차원 그리드 판을 눈으로 그려보듯 직관적으로 배치할 수 있게 해주는 속성이다.

이 프로퍼티를 사용하면 줄(Grid Line) 번호를 계산할 필요 없이, 시각적으로 레이아웃 구조를 설정할 수 있어서 매우 편리하다.

Grid 컨테이너에 직접 지정하며, 기본값은 none이다.

ℹ️ Grid Area는 여러 Grid Cell로 구성된 영역이다.

Syntax

Values

문자열로 Grid Area의 이름을 지정한다. 이름으로 쓰일 값을 토큰이라 하는 모양. . 토큰은 이름 없는 빈 Grid Cell을 의미한다.

⚠️ grid-area 없이 grid-template-areas만 단독으로 쓰면, 셀이 하나로 합쳐지지 않고 기본 규칙(Auto-placement)에 따라 아이템이 셀을 하나씩 차례대로 차지한다.

문법 오류

grid-template-areas의 각 문자열은 한 행을 의미하며, 모든 행은 같은 개수의 토큰을 가져야 한다.

같은 이름으로 연결된 Grid Area는 반드시 직사각형이어야 하며, L자나 T자 형태처럼 끊어진 영역은 유효하지 않다.
