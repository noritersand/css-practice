display: grid

개요

display: grid는 CSS 그리드 레이아웃(grid layout) 모듈을 활성화하는 프로퍼티로, 그리드 레이아웃은 엘리먼트들을 2차원 그리드로 배치하는 데 사용한다.

플렉스박스보다 제공되는 기능이 더 많고 강력하다. 플렉스박스가 그냥 커피면 그리드는 티오피
