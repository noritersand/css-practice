flex-grow

개요

flex-grow는 플렉스박스 레이아웃에서 플렉스 아이템이 사용 가능한 남은 공간을 자동으로 채울 때, 다른 아이템에 비해 얼마나 더 차지할지를 결정한다.

값이 0이면 추가 공간을 차지하지 않고 지정된 크기를 유지한다. 같은 플렉스 컨테이너 내에서 항목들의 grow 값이 다르면, 다르게 지정된 만큼 비율적으로 더 차지하려고 한다.

반대 개념의 프로퍼티로 flex-shrink가 있다.

🚨이 프로퍼티는 플렉스 컨테이너가 아니라 플렉스 아이템에 지정한다.

Values

flex-grow 값은 음수를 제외한 number 타입으로 지정할 수 있다. 0.1 같은 양의 실수도 가능. 기본값은 0이다.

값이 0일 때

값을 생략하거나 0으로 지정하면 남는 공간이 있어도 원래의 크기를 유지한다.

안커짐

width 프로퍼티를 함께 사용하는 경우

width와 flex-grow가 함께 사용되면 width는 flex-basis를 대신하여 기본 크기로 작동한다. 아이템의 초기 크기는 width의 의해 결정되지만, 공간이 남는 경우 flex-grow에 의해 비율적으로 커진다.
