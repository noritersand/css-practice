flex-shrink

개요

flex-shrink는 플렉스박스 레이아웃에서 플렉스 아이템이 자동으로 줄어들 때 얼마나 비율적으로 줄어들지를 결정하는 프로퍼티다.

모든 플렉스 아이템이 1인 경우 동일한 비율로 줄어들고, 특정 아이템의 shrink 값이 다른 아이템보다 더 크면, 줄어들 때 비율적으로 더 많이 줄어든다. 0으로 설정하면 공간이 부족해도 자동으로 줄어들지 않는다.

반대 개념의 프로퍼티로 flex-grow가 있다.

🚨이 프로퍼티는 플렉스 컨테이너가 아니라 플렉스 아이템에 지정한다.

Values

flex-shrink의 값은 음수를 제외한 number 타입으로 지정할 수 있다. 0.1 같은 양의 실수도 가능. 기본값은 1이다.

width와 함께 사용하는 경우

width와 flex-shrink가 같이 지정되면 width는 flex-basis를 대신하여 기본 크기로 작동한다. 아이템의 초기 크기는 width의 의해 결정되지만, 공간이 부족한 경우 flex-shrink에 의해 비율적으로 줄어든다.
